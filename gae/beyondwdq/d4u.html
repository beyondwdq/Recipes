"Download for you"(d4u) downloads the file from the url specified in the following text field and zip it. You can then download the zip file. This is useful when some file types (typically .torrent file) are block by your network administrator.

Limitations:

  • Only HTTP and HTTPS are supported.
  • The zip file size should be less than 1M bytes.
  • Only ASCII characters are allowed in the file name. Otherwise you will have to specify the file name in the "Save as" text field and only file names of ascii code are allowed. This constraint is caused by a bug in the zipfile module with Python 2.5, which is the Python version supported by GAE. The problem is solved in Python 2.6.5.
